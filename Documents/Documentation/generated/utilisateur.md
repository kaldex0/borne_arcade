# Documentation utilisateur

Généré le 2026-02-09 09:43:25

## Démarrage

- La borne démarre automatiquement sur le menu
- Patientez 10–15 secondes si nécessaire

## Navigation

- Joueur 1 : joystick (haut/bas) pour naviguer
- Bouton A (J1) : lancer le jeu
- Bouton Z (J1) : quitter le menu (confirmation)

## Commandes par défaut

- Joystick J1 : flèches haut/bas/gauche/droite
- Joystick J2 : o / l / k / m
- Boutons J1 : r t y / f g h
- Boutons J2 : a z e / q s d

> Note : L'encodeur de la borne peut être mal câblé. Le mapping réel peut différer. Voir le fichier `borne`.

## Pendant un jeu

- Suivre les commandes affichées dans la zone description
- Pour quitter un jeu, utiliser la touche/commande prévue par le jeu

Exemple (StarDodger) :
- Joystick J1 : déplacer le joueur
- Bouton A : boost court

## Extinction

- Après sortie du menu, la borne s’éteint automatiquement (compte à rebours de 30s)
